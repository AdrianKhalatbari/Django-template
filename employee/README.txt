This app is a Django CRUD (Create Read Update Delete) Example.
Address: https://www.javatpoint.com/django-crud-application
The steps are:
1- Create a Model in models.py
2- Create a Form in forms.py
3- Create View in views.py (Write CRUD process here)
4- Add Urls in urls.py
5- Create templates for show,add and edit objects (index.html, show.html, edit.html)
6- Create CSS file 
7- Makemigrations
8- add new application to settings.py
9- Migrations
